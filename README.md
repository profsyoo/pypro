# pypro

Python programming course repository for electrical engineering students.

## Course goals

By the end of this course, students should be able to:
- Write clear, modular Python code
- Analyze and visualize engineering data
- Use numerical tools for electrical engineering problems
- Build reproducible computational workflows

## Repository structure

- `/syllabus` - course outline, schedule, policies
- `/setup` - environment setup guides and requirements
- `/lessons` - lecture notes, examples, and notebooks
- `/labs` - guided lab activities
- `/assignments` - homework and take-home tasks
- `/projects` - mini-project and final project materials
- `/resources` - reference material and external links

## Suggested weekly flow

1. Read lesson notes in `/lessons`
2. Complete lab work in `/labs`
3. Submit assignment from `/assignments`
4. Apply concepts in `/projects`

## Getting started

1. Install Python 3.10+.
2. Create and activate a virtual environment.
3. Install course packages (listed in `/setup` when provided).
4. Open lesson material and begin with Week 1.

## For instructors

- Put schedule and policy details in `/syllabus`
- Add weekly content under `/lessons`
- Keep assignments and solution keys organized by week
- Update this README when structure or workflow changes

# 파이썬 프로그래밍 (Python Programming)

전기공학 전공자를 위한 파이썬 프로그래밍 기초 및 실무 활용 과정입니다. 본 교과목은 이론 학습과 오프라인 실습의 하이브리드 형태로 진행되며, 최종적으로 생활속 (전기공학적) 문제를 해결하는 **Term Project**를 수행합니다.

---

## 📅 강의 운영 방식

본 교과목은 매주 **2개 차시**로 나누어 진행됩니다.

### [1차시] 온라인 콘텐츠 학습 & 기초 다지기 (75분)
* **동영상 콘텐츠 (40분):** 핵심 이론 및 문법 설명
* **온라인 과제 (25분):** 배운 내용을 확인하는 가벼운 코딩 과제 (1회)
* **퀴즈 (10분):** 개념 점검을 위한 퀴즈 2문항 (문항당 5점, 총 10점)

### [2차시] 오프라인 실습 & 개별 지도 (75분)
* **장소:** 전공 실습실
* **내용:** 1차시에 학습한 이론을 연계·확장한 **실습 과제** 수행
* **지도:** 담당 교수([profsyoo](https://github.com/profsyoo))의 1:1 개별 피드백 및 질의응답
